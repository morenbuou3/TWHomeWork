## Effective Presentation
 - 向听众展示热情，并与之互动
 - 重点关注听众的需求，了解他们需要知道什么
 - 简洁明了，注意力要放在核心信息上
 - 微笑，并与听众产生眼神交流
 - 使用讲述故事的方式
 - 用声音的变化去感染听众
 - 利用你的肢体语言
 - 放松，深呼吸，享受其中
 - 充分的准备
